Configuration loading for zk-lsp.

Configuration is merged from two files in order (project overrides user):

PathScope
$XDG_CONFIG_HOME/zk-lsp/config.tomlUser-level
<wiki-root>/zk-lsp.tomlProject-level

See ZkLspConfig for the full list of recognised keys and WikiConfig for runtime path resolution.
